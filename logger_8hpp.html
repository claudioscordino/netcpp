Enumeration Type Documentation

enum severity_level_t

Severity level.

This severity level can be specified for both console messages and file messages. It allows to disable messages (i.e., OFF), show only error messages (i.e. ERROR), show error and warning messages (i.e. WARN) or show all messages (i.e. DEBUG).
