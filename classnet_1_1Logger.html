Detailed Description

Simple logger to log messages on file and console.

This is the implementation of a simple logger in C++. It is implemented as a Singleton, so it can be easily called through two DEBUG macros. It is Pthread-safe. It allows to log on both file and screen, and to specify a severity level for both.

Example of configuration of the Logger: *

DEBUG_CONF("outputfile", DEBUG, ERROR);

Example of usage of the Logger:

DEBUG(DEBUG, "hello " << "world");
